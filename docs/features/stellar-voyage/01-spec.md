# Feature Spec: Stellar Voyage — 우주 탐험 & 외계생명체 수집 (v1 코어 루프)

**Status:** Confirmed
**Created:** 2026-06-11
**Author:** (brainstorm session)

---

## Overview

시드 기반으로 무한 생성되는 3D 은하를 항행하며, 절차적으로 생성된 행성을 탐사하고
생명체 행성에서 외계생명체를 발견·수집하는 캐주얼 웹 게임.
핵심 보상 루프는 **이동 → 발견 → 수집 → 기록**이며, 유한한 종족 도감 완성이 장기 목표다.

## User Goals

- 플레이어로서, 광활한 3D 은하를 자유롭게 항행하며 **미지의 별계를 발견**하고 싶다
- 플레이어로서, 행성을 탐사해 **희귀한 외계생명체를 수집**하고 도감을 완성하고 싶다
- 플레이어로서, 내가 **방문한 곳과 발견한 것의 기록**을 언제든 돌아보고 싶다
- 플레이어로서, 내 우주의 **시드를 공유**해 친구가 같은 우주를 탐험하게 하고 싶다

## Behavior

### Happy Path

1. **게임 시작** — 첫 실행 시 우주 시드가 생성되고(또는 직접 입력), 시드가 정한 시작 별계에서 게임이 시작된다
2. **은하 뷰** — 3D 은하수가 펼쳐진다. 별들 사이를 카메라로 둘러보고(회전/줌/팬), 이동 가능한 별을 선택하면 별 정보(이름, 분광형, 방문 여부)가 표시된다
3. **워프** — '항행' 실행 시 워프 연출이 재생되고 즉시 해당 별계에 도착한다. 방문 기록이 저장된다
4. **태양계 뷰** — 중심 항성과 궤도를 도는 행성들이 3D로 보인다. 행성은 시드로부터 절차 생성된다(암석형/가스형, 낮은 확률로 생명체 행성). 행성 선택 시 행성 정보 패널이 열린다
5. **행성 탐사** — 생명체 행성에서 '탐사' 실행 → 스캔 연출(기대감 빌드업) → **외계생명체 조우 카드** 공개. 희귀도에 따라 연출이 차별화된다
6. **수집** — 조우한 외계인이 도감에 등록된다. 종족(아키타입)은 최초 발견 시 도감 슬롯이 해금되고, 개체(파츠/색상 변형)는 컬렉션에 추가된다
7. **도감 / 탐사 일지** — 언제든 도감(종족 완성률, 수집 개체 목록)과 일지(방문한 별/행성 타임라인)를 열람할 수 있다

### Error Cases

- **WebGL 미지원 브라우저**: 게임 진입 차단 + 지원 브라우저 안내 화면 표시
- **IndexedDB 사용 불가(사생활 보호 모드 등)**: 메모리 저장으로 폴백하여 플레이는 가능하되, "이 세션의 기록은 저장되지 않습니다" 경고 배너 상시 표시
- **저장 쓰기 실패**: 자동 재시도 후 실패 시 토스트로 알림. 게임 진행은 차단하지 않는다
- **유효하지 않은 시드 입력**: 입력 단계에서 검증하고 인라인 에러 표시. 게임 상태는 변경되지 않는다

### Edge Cases

| 상황 | 기대 동작 |
|------|-----------|
| 태양계에 생명체 행성이 없음 | 정상 상황(약 90%). 행성 관측 정보와 방문 기록만 남는다 |
| 이미 탐사한 생명체 행성 재방문 | 동일한 외계인 개체가 다시 보인다(시드 결정론). "이미 조우함" 표시, 중복 등록 없음 |
| 같은 종족을 다른 행성에서 발견 | 종족 도감은 이미 해금 상태, 새 **개체**(다른 파츠/색 조합)로 컬렉션에 추가 |
| 시드 공유 후 친구가 같은 좌표 방문 | 동일한 별/행성/외계인 개체가 생성된다(좌표 공유 = 발견 공유) |
| 현재 머무는 별을 다시 선택 | '항행' 비활성화, 태양계 뷰로 바로 진입 |
| 데이터 삭제/새 기기 | 기록은 사라지나, 같은 시드를 입력하면 같은 우주를 처음부터 재탐험 가능 |
| 저사양 기기/모바일 | 별 입자 수·행성 텍스처 품질을 자동 하향. 터치 제스처(드래그 회전, 핀치 줌) 지원 |
| 탭 비활성화 | 렌더 루프 일시정지(배터리/CPU 보호), 복귀 시 재개 |

## Interface Design

### 화면 구성

| 화면 | 역할 | 핵심 인터랙션 |
|------|------|---------------|
| 은하 뷰 (Galaxy) | 3D 은하수, 별 선택과 항행 | 카메라 회전/줌/팬, 별 선택, 워프 실행 |
| 태양계 뷰 (System) | 항성 + 궤도 행성들 | 행성 선택, 행성 정보 패널, 은하 뷰 복귀 |
| 탐사/조우 (Encounter) | 스캔 연출 → 외계인 카드 공개 | 탐사 실행, 카드 확인, 도감 바로가기 |
| 도감 (Codex) | 종족 슬롯 그리드 + 수집 개체 목록 | 종족/개체 상세, 완성률 표시, 미발견 종족은 실루엣 |
| 탐사 일지 (Journal) | 방문 타임라인 | 별/행성 기록 열람, 시드 복사·공유 |

### Data Model (개념 수준)

- **Universe**: `seed` 하나로 전체가 결정. 별/행성/외계인은 좌표·인덱스에서 결정론적으로 파생 — **생성물은 저장하지 않는다**
- **SpeciesArchetype** (정적 데이터, 약 60종): `id`, `name`, `rarity(common|rare|epic|legendary)`, 파츠 구성 규칙, 도감 설명
- **AlienIndividual**: `speciesId` + 파츠/색상 변형 값 + 발견 위치/시각 — 행성 좌표에서 결정론적으로 생성
- **VisitRecord**: 방문한 별/행성 좌표, 방문 시각, 탐사 여부
- **CollectionEntry**: 수집한 개체, 최초 발견 여부
- 저장 대상은 **플레이어 기록(VisitRecord, CollectionEntry)과 설정뿐**이다

### 핵심 수치 (기본값, 밸런싱으로 조정 가능)

| 항목 | 기본값 |
|------|--------|
| 종족 수 | 60종 (커먼 32 / 레어 18 / 에픽 8 / 레전더리 2) |
| 태양계당 행성 수 | 1~8개 |
| 생명체 행성 출현율 | 약 10% |
| 종족 출현 가중치 | 커먼 70% / 레어 22% / 에픽 7% / 레전더리 1% |
| 생명체 행성당 외계인 | 1개체 (시드 고정) |

## Acceptance Criteria

- [ ] WHEN 플레이어가 처음 게임을 실행하면 THE SYSTEM SHALL 시드를 생성하고 시작 별계의 태양계 뷰를 표시한다
- [ ] WHEN 플레이어가 은하 뷰에서 별을 선택하고 항행을 실행하면 THE SYSTEM SHALL 워프 연출 후 해당 별계의 태양계 뷰를 표시하고 방문 기록을 저장한다
- [ ] THE SYSTEM SHALL 동일한 시드와 동일한 좌표에 대해 항상 동일한 별·행성·외계인 개체를 생성한다
- [ ] WHEN 플레이어가 생명체 행성에서 탐사를 실행하면 THE SYSTEM SHALL 스캔 연출 후 외계인 조우 카드를 표시하고 컬렉션에 등록한다
- [ ] WHEN 미발견 종족을 최초 조우하면 THE SYSTEM SHALL 도감의 해당 종족 슬롯을 해금하고 최초 발견 표시를 한다
- [ ] WHEN 이미 탐사한 행성을 재방문하면 THE SYSTEM SHALL 동일 개체를 표시하되 중복 등록하지 않는다
- [ ] WHILE 도감 화면이 열려 있는 동안 THE SYSTEM SHALL 종족별 발견 상태(미발견 실루엣 포함)와 전체 완성률을 표시한다
- [ ] WHEN 페이지를 새로고침하거나 브라우저를 재시작하면 THE SYSTEM SHALL IndexedDB에서 방문 기록·컬렉션·현재 위치를 복원한다
- [ ] IF WebGL을 사용할 수 없으면 THEN THE SYSTEM SHALL 게임 진입을 차단하고 지원 환경 안내를 표시한다
- [ ] IF IndexedDB를 사용할 수 없으면 THEN THE SYSTEM SHALL 메모리 저장으로 동작하며 기록이 저장되지 않음을 상시 안내한다
- [ ] THE SYSTEM SHALL 데스크탑(마우스/키보드)과 모바일(터치 제스처, 375px+) 양쪽에서 모든 핵심 루프를 수행할 수 있게 한다
- [ ] THE SYSTEM SHALL 도감·일지·정보 패널 등 UI 요소를 키보드로 조작 가능하게 한다

## Scope

**In Scope (v1):**
- 은하 뷰 → 태양계 뷰 → 행성 탐사 → 조우의 전체 코어 루프
- 시드 기반 절차 생성(별, 행성, 외계인), 시드 입력/공유
- 도감(60종) + 탐사 일지, IndexedDB 로컬 저장/복원
- 4단계 희귀도와 등급별 조우 연출 차별화
- 데스크탑 우선 + 모바일 대응(터치, 성능 자동 하향)

**Out of Scope (v1 제외, 향후 검토):**
- 서버/계정/멀티플레이/랭킹/도감 온라인 공유
- 자원 관리(연료, 화폐), 우주선 업그레이드
- 포획 미니게임, 변이체(샤이니) 시스템
- 사운드/BGM, 거래·미션·스토리 콘텐츠
- 절차적 3D 외계인(외계인은 2D 파츠 조합 카드로 고정)
